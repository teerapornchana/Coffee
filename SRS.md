# Software Requirements Specification (SRS)
## ระบบรับจองโต๊ะร้านกาแฟ (Coffee Shop Table Reservation System)

### 1. ภาพรวม (Overview)
ระบบรับจองโต๊ะร้านกาแฟผ่านเว็บ/แอปพลิเคชัน ที่ช่วยอำนวยความสะดวกให้ลูกค้าจองคิวล่วงหน้าได้โดยไม่ต้องสมัครสมาชิก และมีระบบหลังบ้าน (Back-office) ให้พนักงานสามารถจัดการคิว เช็คอิน และดูรายการจองรายวันได้อย่างเป็นระบบ โดยระบบจะจัดการเวลาเป็นรอบ (Time slot) และปิดรับจองในรอบที่คิวเต็มแล้ว

### 2. ผู้ใช้งานหลัก (Users)
1. **ลูกค้าทั่วไป (Customer):** 
   - ต้องการจองคิวเพื่อมาทานที่ร้าน 
   - สามารถจองล่วงหน้าได้ 1-7 วัน 
   - ใช้งานได้ทันทีโดยไม่ต้องเข้าสู่ระบบ (Guest Checkout)
2. **พนักงานร้าน (Staff/Admin):** 
   - ต้องการระบบสำหรับดูคิวในแต่ละวัน 
   - ยืนยันการมาถึงของลูกค้า และจัดการคิวกรณีมีปัญหา

### 3. ฟีเจอร์ที่ต้องมี (Must-have Features)
#### 3.1 ฝั่งลูกค้า (Customer Interface)
- **Time Slot Selection:** ระบบให้เลือกวันและรอบเวลา (เช่น รอบละ 1.5 - 2 ชั่วโมง)
- **Availability Check:** ระบบบล็อกไม่ให้จองในวัน/เวลาที่คิวเต็ม (Fully Booked)
- **Booking Confirmation:** หน้าจอแสดงผลการจองสำเร็จพร้อมรายละเอียดและรหัสการจอง เพื่อให้ลูกค้าแคปหน้าจอเก็บไว้เป็นหลักฐาน

#### 3.2 ฝั่งพนักงาน (Staff Interface)
- **Daily Queue Dashboard:** หน้าจอแสดงรายการคิวจองในแต่ละวัน
- **Status Management:** ปุ่มอัปเดตสถานะคิว (เช็คอินลูกค้าเมื่อมาถึง / กดยกเลิกการจอง)
- **No-show Handling:** ระบบยกเลิกคิว (No-show) ที่พนักงานกดได้เองกรณีลูกค้ามาสายเกิน 15 นาที
- **Manual Table Assignment:** การจัดสรรโต๊ะแบบ Manual (พนักงานหาโต๊ะให้เองเมื่อลูกค้ามาถึง ไม่ต้องผูกเลขโต๊ะในระบบล่วงหน้า)

### 4. ฟีเจอร์ที่น่ามีในอนาคต (Nice-to-have Features)
- **Notifications:** ระบบแจ้งเตือนลูกค้าทาง SMS หรือ LINE (ทั้งตอนจองสำเร็จและแจ้งเตือนเมื่อใกล้ถึงเวลา)
- **Analytics Dashboard:** หน้ารายงานสถิติเพื่อดูช่วงเวลาที่คนแน่นที่สุด (Peak hours) และข้อมูลลูกค้าเพื่อใช้ทำการตลาด

### 5. ข้อมูลที่ต้องเก็บ (Data Dictionary)

| ฟิลด์ข้อมูล (Data Field) | คำอธิบาย (Description) | ชนิดข้อมูล (Type) | ความจำเป็น |
| :--- | :--- | :--- | :--- |
| **ชื่อลูกค้า** | ชื่อของผู้ที่ทำการจองเพื่อใช้เรียกตอนมาร้าน | `String` | Required |
| **เบอร์โทรศัพท์** | ใช้ยืนยันตัวตน ค้นหาคิว และติดต่อกลับ | `String` | Required |
| **วันที่และเวลาจอง** | วันที่และรอบเวลาที่ระบุ (Time Slot) | `Datetime` | Required |
| **จำนวนคน** | จำนวนลูกค้าในกลุ่มเพื่อใช้กะขนาดโต๊ะ | `Integer` | Required |
| **หมายเหตุ** | ความต้องการพิเศษ เช่น เอาเก้าอี้เด็ก, แพ้อาหาร | `Text` | Optional |
| **สถานะการจอง** | Booked, Arrived, No-show, Cancelled | `Enum` | Required |
